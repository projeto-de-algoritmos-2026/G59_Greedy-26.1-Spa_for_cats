# 🐱 Spa Cat - Algoritmos Gulosos (G59)

Este projeto aplica conceitos de **Projeto de Algoritmos** para gerenciar a fila de um Spa de Gatos.

### 🧠 Algoritmos Implementados:
1. **Interval Partitioning:** Controle de acesso baseado em recursos limitados (`SALAS_DISPONIVEIS`).
2. **Earliest Deadline First (EDF):** Ordenação da fila para minimizar o atraso máximo.

### 🚀 Como rodar:
1. **Clone o repositório:** `git clone https://github.com/projeto-de-algoritmos-2026/G59_Greedy-26.1-Spa_for_cats`
2. **Crie o ambiente virtual:** `python3 -m venv venv && source venv/bin/activate`
3. **Instale as dependências:** `pip install -r requirements.txt`
4. **Inicie o servidor:** `python3 main.py`
5. **Acesse:** Abra o arquivo `index.html` no seu navegador.
